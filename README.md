# CSGO-Mod-Xash-Mod
A Counter Strike Global Offensive mod for Xash3D / CSMOE by ChrystaL.
# Features
•Classic HUD on CSGO style

•Option that can change weapons

•10 CSGO knifes

•CSGO style UI,VGUI and Panel

•Running on CSMOE Launcher
